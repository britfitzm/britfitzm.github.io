Age

Users can select from a range of age groups.
The age groups decide the level of difficulty of the games.

Selection

Users have a limited range of games to choose from.
This is so that they are not over-stimulated and will
play one game for longer.

Answers

There is no answer tracking for correct or incorrect.
The games allow for multiple chances to answer the
questions. It also guiedes the user to the answer
with hints about the answer.
